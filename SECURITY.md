# Security Policy

## Supported versions

The latest released version receives security fixes.

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Instead, report them privately via GitHub's
[Security Advisories](https://github.com/khaledalam/unit/security/advisories/new),
or email **khaledalam.net@gmail.com**.

You will receive an acknowledgement within a few days, and we will keep you
informed as the issue is investigated and resolved.
